Lynx
 Demos Sandbox Development Whitepaper
Lynx is a programming language designed to make coding intuitive and efficient so that beginners and experienced programmers alike can write great software. Edits to code are made directly on representations like diagrams and pictures to dramatically reduce the learning curve and. The flexibility of the graphical features are kept on track by a state-of-the-art type system so programmers never have to worry about how their code will break unexpectedly.
